# Nexus Coach Colors

> Colors that embody guidance, structure, and the warmth of a professional coaching relationship.

## Strategic Foundation

**Brand Alignment:** Nexus Coach's color palette reflects The Guide archetype — calm, organized, and empowering — supported by The Architect's structural precision. The teal-anchored palette communicates connection and growth (the bridge between trust and forward momentum), while deep slate tones convey the professional backbone of a well-run practice. Warm amber accents signal action and achievement without urgency. The palette is clean and modern, inspired by confident SaaS brands that let content breathe against structured backgrounds.

## Color Ratios & Hierarchy

### 60-30-10 Rule
- **60% Neutrals:** White, slates — breathing room, readability, and the clean canvas coaches expect
- **30% Primary:** Guide Teal, Structure Slate — brand presence and structural authority
- **10% Accent:** Momentum Amber, secondary colors — emphasis, action, and achievement

### Application Priority
1. **Guide Teal** for primary actions, interactive elements, and brand moments
2. **Momentum Amber** for the most important CTAs and achievement highlights
3. **Structure Slate** for dark backgrounds, headings, and navigation
4. **Neutrals** for backgrounds, body content, and structural elements
5. **Semantic colors** for status and feedback only — never decorative

### Button Hierarchy

## Accessibility

### Contrast Ratios (WCAG 2.1)

| Combination | Ratio | Level |
|-------------|-------|-------|
| Structure Slate on White | 16.7:1 | AAA ✓ |
| Guide Teal on White | 5.5:1 | AA ✓ |
| Slate 600 on White | 5.7:1 | AA ✓ |
| White on Guide Teal | 5.5:1 | AA ✓ |
| White on Structure Slate | 16.7:1 | AAA ✓ |
| Structure Slate on Momentum Amber | 5.3:1 | AA ✓ |
| Momentum Amber on White | 3.2:1 | AA Large ✓ |
| White on Momentum Amber | 3.2:1 | AA Large ✓ |

### Guidelines
- **Body text:** Use Slate 600+ on light backgrounds for AA compliance
- **Large text (18pt+):** Can use 3:1 minimum (allows Momentum Amber on white)
- **Interactive elements:** Minimum AA compliance (4.5:1)
- **CTA buttons:** Use Structure Slate text on Momentum Amber backgrounds for 5.3:1 contrast
- **Don't rely on color alone:** Always pair with icons, text labels, or patterns

## Usage Guidelines

### Do
- Use Guide Teal as the dominant brand color for interactive elements
- Use Structure Slate for dark navigation and high-authority text
- Maintain generous white space — the palette breathes best with room
- Apply Momentum Amber sparingly for maximum CTA impact
- Test all text/background combinations for WCAG AA minimum
- Use the extended teal scale for subtle background treatments
- Use semantic colors exclusively for their intended purpose

### Don't
- Use Momentum Amber for large background fills
- Combine Guide Teal and Insight Indigo in close proximity
- Apply colors at low opacity that compromise contrast
- Use colors outside this system for UI elements
- Default to color alone for conveying meaning
- Override semantic color meanings (e.g., red for success)
- Use bright accent colors for body text
