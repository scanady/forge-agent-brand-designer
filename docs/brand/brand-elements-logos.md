# OpenLife Logos

> Visual representation of liberation, openness, and collaborative innovation.

## Strategic Foundation

**Brand Alignment:** The OpenLife logo embodies The Liberator archetype—visually communicating freedom from vendor lock-in and collaborative power. The mark feels confident yet approachable, modern yet trustworthy for enterprise audiences.

## Logo Concept

### Symbolism
The logo combines:
1. **Openness:** Expansive, unlocked form suggesting transparency
2. **Life/Growth:** Upward motion suggesting vitality and evolution

### Wordmark
- Set in **Inter Bold** (700)
- "OpenLife" as one word
- Optional: "Open" in Liberation Blue, "Life" in Foundation Navy

## Logo Variations

### Primary (Horizontal)
Default for most applications.

| Attribute | Specification |
|-----------|---------------|
| Composition | Mark + Wordmark horizontal |
| Min Width | 120px digital / 1.5" print |
| Aspect Ratio | ~3:1 |

### Stacked (Vertical)
For square or vertical spaces.

| Attribute | Specification |
|-----------|---------------|
| Composition | Mark above wordmark |
| Min Width | 80px digital / 1" print |
| Aspect Ratio | ~1:1.2 |

### Logomark Only
For constrained spaces.

| Attribute | Specification |
|-----------|---------------|
| Composition | Mark only |
| Min Size | 24px digital / 0.25" print |
| Usage | App icons, favicons |

### Wordmark Only
When mark is established elsewhere.

| Attribute | Specification |
|-----------|---------------|
| Composition | "OpenLife" text |
| Font | Inter Bold |
| Usage | Footers, co-branding |

## Clear Space

Minimum clear space equals the height of the "O" on all sides.

```
    ┌─────────────────────────┐
    │         O height        │
    │   ┌─────────────────┐   │
    │ O │     LOGO        │ O │
    │   └─────────────────┘   │
    │         O height        │
    └─────────────────────────┘
```

### Digital

| Variation | Min Width | Min Height |
|-----------|-----------|------------|

## Color Variations

### Full Color (Primary)
- Mark: Liberation Blue (`#0066CC`)
- Wordmark: Foundation Navy (`#0D1B2A`)
- Background: White or light neutrals

### Full Color on Dark
- Mark: Liberation Blue Light (`#3B82F6`) or White
- Wordmark: White
- Background: Foundation Navy or dark

### Monochrome Dark
- All elements: Foundation Navy
- Background: Light surfaces

### Monochrome Light (Reversed)
- All elements: White
- Background: Dark surfaces, photography

## Background Usage

| Background | Logo Version |
|------------|--------------|
| White | Full Color |
| Gray 50–100 | Full Color |
| Liberation Blue | Reversed |
| Foundation Navy | Reversed |
| Photography (light) | Full Color |
| Photography (dark) | Reversed |

Ensure 4.5:1 minimum contrast.

## Incorrect Usage

### Do Not
- Stretch or distort proportions
- Rotate at angles
- Add shadows, glows, or effects
- Change logo colors
- Place on busy backgrounds
- Rearrange mark and wordmark
- Recreate with other fonts
- Add outlines or strokes
- Violate clear space
- Use below minimum size
- Apply transparency
- Animate without approval

### Naming Convention
```

Examples:
openlife-logo-primary-fullcolor.svg
openlife-logo-stacked-reversed.png
openlife-logo-mark-monochrome.svg
```

## Co-Branding

When appearing with partners:
- Equal visual weight
- Clear divider between logos
- Consistent color treatment
- OpenLife first (left) in horizontal

## Accessibility

- Alt text: "OpenLife"
- Linked: "OpenLife - Go to homepage"
- Decorative: Empty alt or aria-hidden
- Ensure contrast meets WCAG AA

## Usage Guidelines

### Do
- Use approved variations only
- Maintain clear space
- Ensure proper contrast
- Use correct file formats
- Follow minimum sizes

### Don't
- Modify the logo
- Use outdated versions
- Apply effects
- Place on conflicting backgrounds
- Crop the logo
